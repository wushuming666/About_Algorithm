## 大一
懵懵懂懂学习编程，初学的时候很痛苦，大一上曾经一个下午也没能写出水仙花数。
大一的时候了解到了数学建模，参加过不咋莫严厉的集训。和队友在东七熬过通宵写小论文、与小级和阿鑫一起参赛了2021年的数学建模，拿了省二。不得不承认我挺混的，谢谢队友、谢谢老师们。
## 大二

写这个的时候处于蓝桥杯省赛结束后的两星期左右。想了一年的事情做了，转项目了。
